# Fake News Detector

A fine-tuned DistilBERT model that classifies news articles as Real or Fake.

## Model
- Base: `distilbert-base-uncased`
- Dataset: WELFake (10,000 samples)
- Accuracy: 99%
- F1 Score: 0.99

## Tech Stack
- Hugging Face Transformers
- PyTorch
- Gradio
- Google Colab (training)
